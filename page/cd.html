What is CD?

CD stands for Controlled Difficulty and it is a custom game mode that is specifically designed to control many aspects of the base game. While its intended purpose is to remove the randomness of Killing Floor 2 and present a harder experience than Hell on Earth, CD can be configured for easier scenarios.

There are 8 main ‘variables’ that can affect the difficulty of a CD match: spawn poll, cohort size, max monsters, spawn cycle, max monsters, wave size fakes, large hp fakes and map.


Spawn Poll & Cohort Size

Spawn Poll and Cohort Size determine the amount of Zeds that are spawning on the map. Spawn Poll is the amount of time in seconds before another batch of Zeds can spawn and Cohort Size is the amount of Zeds per batch.

Together they create something called Zeds Per Second which is calculated by Cohort Size/Spawn Poll. Ex. 5 Zeds / 2 Seconds = 2.5 Zeds per Second.

Note:This feature is typically set and locked on Tamari’s Servers. So don’t worry about this too much if you are playing there.

Spawn Cycle

A good analogy to describe a Spawn Cycle is a recipe. It is a set of instructions to spawn the same type of Zed and the specific amount of them. This is to make more consistent gameplay.

More info on Spawn Cycles HERE.

Max Monsters

Max Monsters determine the max amount of Zeds that can be present on the map. Higher the value, the more difficult the game is. Default value in Killing Floor 2 is 32.

Wave Size Fakes

In a nutshell, Wave Size Fakes determine the amount of Zeds that spawn per wave. By default, the value of Wave Size Fakes is 0 (which scales with the amount of players present). However, if Wave Size Fakes is set to 12, the amount of Zeds that spawn per wave would be like if 12 players were present (regardless of player count).

Large HP Fakes

Large HP Fakes function similarly to Wave Size Fakes, but determine the amount of health that large Zeds have. The standard for Large HP fakes would be 6.
